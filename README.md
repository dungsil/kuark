# Kuark
순수 코틀린으로 작성된 유틸리티 라이브러리

## "Kuark" ?
Kuark(IPA: /kwɔːk/, same as 'quark')는 물질을 가장 기본적인 입자인 쿼크(Quark)와 코틀린(Kotlin™)의 첫글자를 딴 이름으로
간단하고 가볍지만 유용한 유틸리티를 목적으로 합니다.
